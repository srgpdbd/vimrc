# Just my vimrc

Try to edit this

## Plugins inside

- Color schemes
- YouCompleteMe
- NERDTree (on F3)
- FZF

## How to run

- cp .vimrc ~/
- Open vim and do **:BundleInstall**
- cd ```~/.vim/bundle/YouCompleteMe && python3 install.py``` **(requires python3 and make)**
- cd ```~/.vim/bundle/markdown-preview.nvim/app && ./install.sh```
